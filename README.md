# ES6 React

> Apenas um show


## Required

You need install `jspm`

```
$ npm i -g jspm
```


## Recommend

You can install `ws`

```
$ npm i -g local-web-server
```


## Install

```
$ npm i
$ jspm i
```


## Run

```
$ grunt serve
```


### Build

```
$ make build
```
